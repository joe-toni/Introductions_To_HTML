Introductions

Welcome to the website, here you will find a very brief explanation of many but not all HTML elments available to you. We purposefully left out any elements that are obsolite or depreciated, as well as any root and metadata elements. The focus here was to give users a quick glance at some of the more active elements, the ones meant to provide page structure or aid in some functionality. With that in mind the elements have been divided into four groups, sectioning, text, emebeded content, and forms. Explanations on each of these groupings is provided at their respective introductions. As an additional aid to understanding the content a user can view simple examples using the testing area below. Simply navigate to the desired element and click on the view example button, the textarea will then be populated with an example for the element currently being viewed while the frame next to it renders the result along with any changes made. Give it a try for an introductory greating.
